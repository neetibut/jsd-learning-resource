# Link Maintenance

Run this checklist before each cohort. A dead "Go Deeper" link hits students when
they're most engaged. Takes about 20 minutes for the full curriculum.

## How to check

Open each link. If it 404s or the content has changed significantly, update or
replace the link in the topic README before sharing with students.

Good replacements for dead video links: search YouTube for the same topic by a
reliable creator (Traversy Media, Fireship, Kevin Powell for CSS). Prefer channels
that have been active for 3+ years.
